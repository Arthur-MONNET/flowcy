# Agent D - Frontend / Vue + TS + Tailwind (UI)

## Mission
Build the app UI with clear components, consistent design patterns, and responsive behavior.

## Primary Responsibilities
- Define Vue structure (Vite + Pinia + Router).
- Build layout (sidebar, filters, layer toggles).
- Establish UI patterns (cards, chips, badges, skeletons).
- Ensure accessibility and performance.

## Deliverables
- Component structure in src/components.
- Layout and responsive behavior.
- UI pattern catalog notes (ARCH.md or SPEC.md).

## When to Engage
- Any UI design or UX decision.
- When adding new panels or interactions.

## Inputs
- Product needs from Agent A.
- Data contracts from Agent F.

## Outputs
- Vue components aligned with contracts.

## Guardrails
- UI must not invent data fields.
- Keep components small and composable.
